# 2. 操作组件规则

## 2.1 normal\_list\_page\_parser\_list\_page\_parser\_use\_xpath 组件

### 2.1.1 参数表

| 参数                | 参数定义标识ID | 参数名称                | 参数Value值                                                                | 备注                    |
| ----------------- | -------- | ------------------- | ----------------------------------------------------------------------- | --------------------- |
| list\_xpath\_exp  | 51       | 列表xpath规则           | 例：//div\[@class='elementor-loop-container elementor-grid']/div          |                       |
| link\_xpath\_exp  | 52       | 列表xpath规则链接规则       | 例：//a\[1]/@href                                                         |                       |
| title\_xpath\_exp | 53       | 列表xpath规则标题规则<br /> | 例：//h3\[@class='elementor-heading-title elementor-size-default']/text() |                       |
| list\_page\_type  |          | 列表页类型               | {1，2}                                                                   | 1:仅列表抓取<br />2:列表详情抓取 |

### 2.1.2 示例配置

```json
{
    "op_name": "普通列表解析列表列表xpath解析",
    "op_type": "normal_list_page_parser_list_page_parser_use_xpath",
    "op_sequence": 1,
    "op_param": [
        {
            "op_type": "normal_list_page_parser_list_page_parser_use_xpath",
            "param_define_id": 51,
            "param_code": "list_xpath_exp",
            "param_name": "列表xpath规则",
            "param_value": "//div[@class='elementor-loop-container elementor-grid']/div"
        },
        {
            "op_type": "normal_list_page_parser_list_page_parser_use_xpath",
            "param_define_id": 52,
            "param_code": "link_xpath_exp",
            "param_name": "列表xpath规则链接规则",
            "param_value": "//a[1]/@href"
        },
        {
            "op_type": "normal_list_page_parser_list_page_parser_use_xpath",
            "param_define_id": 53,
            "param_code": "title_xpath_exp",
            "param_name": "列表xpath规则标题规则",
            "param_value": "//h3[@class='elementor-heading-title elementor-size-default']/text()"
        }
    ],
    "sub_op_list": null,
    "spider_type": 9,
    "list_page_type": 1
}
```
