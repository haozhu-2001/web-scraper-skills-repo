```json
  "op_name": "data.extract",

    "target": "item",
    "field": {
      "view": {
        "selector": "xpath=//div[@class='extra-info-list']/span[3]/text()",
        "processors": [
          {
            "type": "regex_extract",
            "pattern": "(\\d+(?:\\.\\d+)?\\s*[万亿kKmM]?)",
            "group": 1 #group默认为1，与re.regex同逻辑
          }
        ]
      },
      "html_content": {
        "selector": "xpath=//html"
      },
      "title": "xpath=//div[@class='item-text-content-title']/text()"
    }
  }
}
```

#### 2.3.1.1 参数表

| 参数            | 参数类型  | 是否必须 | 默认值      | 取值范围                         | 备注                                                                                                                                                                                                                                                                              |
| ------------- | ----- | ---- | -------- | ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| pre\_delay    | float | 否    | 0        | >0                           | 步骤执行前延迟(s)                                                                                                                                                                                                                                                                      |
| post\_delay   | float | 否    | 0        | >0                           | 步骤执行后延迟(s)                                                                                                                                                                                                                                                                      |
| step\_timeout | float | 否    | 15       | >0                           | 单步骤超时时间(s)：寻找稳定的可点击目标最长等待时间                                                                                                                                                                                                                                                     |
| source        | str   | 是    | "page"   | 与foreach操作loop\_item一致<br /> | source和open\_mode为一组参数，source表示执行的项，open\_mode为打开详情页的方式<br />详情页跳转类：<br />1. source + open\_mode + selector 点击source内的selector 实现跳转详情页<br />2. source + open\_mode 点击source实现跳转详情页<br />普通点击按钮类：<br />1. source + selector 点击source内的selector 按钮<br />2. selector 点击selector 按钮 |
| open\_mode    | str   | 是    | new\_tab | {"new\_tab"}                 |                                                                                                                                                                                                                                                                                 |
| selector      | str   | 是    | 无        | 参照配置规则表                      | [ 选择器（selector）配置规则表](https://prnewswire.feishu.cn/wiki/VhSuwnjsJisUuskrwxrcceyBnPf)                                                                                                                                                                                            |

#### 2.3.1.2 示例配置

```json
// 详情页跳转类：
// 1. source + open_mode + selector 点击source内的selector 实现跳转详情页
{
    "op_sequence": 1,
    "op_name": "element.click",
    "op_type": "element.click",
    "params": {
      "source": "row",
      "open_mode": "new_tab",
      "selector":"xpath=//div[@class='tit']/h3/a"
    }
}
// 2. source + open_mode 点击source实现跳转详情页
{
    "op_sequence": 1,
    "op_name": "element.click",
    "op_type": "element.click",
    "params": {
      "source": "row",
      "open_mode": "new_tab"
    }
}
```

```json
// 普通点击按钮类
// 1. source + selector 点击source内的selector 按钮
{
    "op_sequence": 1,
    "op_name": "element.click",
    "op_type": "element.click",
    "params": {
        "source": "row",
        "selector": "xpath=//button[normalize-space()='链接']"
    }
```
